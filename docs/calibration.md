# Calibration Guide

## Why Calibration Matters

Low-cost particulate sensors are useful for monitoring trends, but they are not equivalent to certified regulatory instruments.

Sensor behavior can be influenced by:

- warm-up time
- airflow conditions
- humidity
- enclosure design
- particle composition
- long-term drift

## Practical Calibration Notes

1. Allow the sensor to warm up before relying on measurements
2. Compare readings against a known reference instrument if possible
3. Record readings across multiple environmental conditions
4. Look for repeatability, stability, and obvious outliers
5. Recalibrate expectations if the deployment enclosure changes airflow

## Notes

This prototype is best understood as a monitoring and educational system rather than a compliance-grade measurement device.
